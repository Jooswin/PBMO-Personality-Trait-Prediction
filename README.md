**PBMO for Personality Trait Prediction**


## 1. Create a Conda Environment

Open your terminal and run the following command to create a new Conda environment named `pbmo_personality_trait` with Python 3.10:

`conda create -n pbmo_personality_trait python=3.10`

## 2. Activate conda Virtual Environment

`conda activate pbmo_personality_trait`

## 3. Install Required Packages

run: `pip install -r requirements.txt`

This will install all necessary packages for the project.

## 4. Run the Project

Once the environment and packages are ready, run the main program:

`python main.py`

**note:** Make sure you are in the project directory where main.py is located.
